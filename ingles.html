Project Description

This project contains the activities developed and the knowledge acquired in the semester in the subjects of Operating Systems, Introduction to hardware and programming logic, integrating them into a web application that allows the visualization of the topics seen through the content requested in each subject, to This web application has a menu with the respective submenus for better organization and a clear structure.
question
The integrative Project tries to make an application which is a menu, in which the information requested by each subject is evident, each subject will have its respective submenu in which we will load work of each subject, the subjects that make up the program are English, operating systems, introduction to hardware and programming logic, which is the core subject .
How did you do?
The group first defined what was request as a Project, second we looked for information and together with the teachers we learned from scratch how to start programming, for example: design our own website, create codes, etc. With the knowledge acquired, we create our website, then in this implemented the menu and submenu, we learned in programming logic class, along with request videos for Operating system and Introduction to Hardware. As We said before, most of the information we obtained was from the classes and teacher at the university, however in the case of the website, Steven had prior knowledge of HTLM to make the page, and the questions or errors that we had in the creation of the code, we solved them by searching in the internet or watching videos.
What for you did this Project?
The main objetive for us when doing this Project is to test the knowlegde acquired during the first semester of system engineering, this through trial and error. Resulting in a website with was requested in each class.
